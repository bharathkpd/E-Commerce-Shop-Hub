# Shop Hub

A simple e-commerce front-end project built using **HTML**, **CSS**, and **JavaScript**.

## Features
- Responsive design  
- Product listing page  
- Add to cart (UI only)  
- Clean and minimal interface  

## Files
- `index.html`  
- `style.css`  
- `script.js`  
- `assets/` (images)

## How to Run
Just open **index.html** in any browser.

## Tech Used
HTML, CSS, JavaScript.
